## Description

Mock venue and event are seeded on the first run.

Endpoint: `POST localhost:3000/reservations`

Ticket ids for manual testing:

```
eventId: 1
row1: [1, 2, 3, 4, 5, 6]
row2: [7, 8, 9, 10, 11, 12]
```

## Installation

```bash
# Install dependencies
$ yarn

# Run app
$ yarn start

# Watch mode
$ yarn start:dev
```

## Test

```bash
yarn test
```
